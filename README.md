# 项目简介

## 核心功能
- 基于YOLOv8的太阳能板缺陷检测
- 支持4类缺陷检测：掉落、破裂、生锈、面板
- 支持多种输入源：单张图片、图片文件夹、视频文件、本地摄像头
- 提供实时显示功能：目标位置、目标总数、置信度
- 支持自定义检测参数：置信度阈值、IOU阈值
- 支持模型选择和显示选项
- 批量检测功能，可查看所有检测结果
- 支持检测结果保存

## 适用场景
- 太阳能板生产质量检测
- 太阳能电站维护检查
- 光伏组件缺陷识别
- 工业自动化检测系统

## 技术特点
- 使用Pycharm+Anaconda或VSCode+Anaconda开发环境
- 基于Python 3.9，依赖opencv-python、PyQt5、ultralytics、torch1.9等技术栈
- 包含6400张标注数据集
- 提供完整UI界面，操作简便
- 已训练好模型，可直接使用
